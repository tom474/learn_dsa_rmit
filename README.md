- Java

- 04 - Non-linear Structures
- 05 - Time and Space Trade-offs
- 06 - Brute Force
- 07 - Divide and Conquer
- 08 - Decrease and Conquer
- 09 - Transform and Conquer
- 10 - Greedy and Dynamic Programming
